footer tag:
-----------
	This is important for SEO because it provides additional information about the website and its content. It can contain links to related pages and social media profiles.
	<footer> 
		<p>Copyright © YourWebsite.com</p> 
		<a href="https://www.facebook.com/YourWebsite">Facebook</a> 
		<a href="https://twitter.com/YourWebsite">Twitter</a> 
	</footer>
