what it the use of throw block:
---------------------------------
throw passes the error to the catch block.

    try {
        let age = 25;
        if(age < 18) {
            throw new Error("under Age");
        } else {
            console.log("registration successul");
        }
    }
    catch (err) {
        console.log(err.message);
    }

Error handling using try and catch block:
-----------------------------------------

    1️⃣ Synchronous Code
        try {
            let data = value + 10;
            // value is not defined so it will throw an error. Handle it using catch block
        } 
        catch (err) {
            console.log("undefined variable", err.message);
        }


    2️⃣ Asynchronous with setTimeout

        ⚠️ Normal try/catch doesn’t work inside async callbacks.

        try {
            setTimeout(() => {
                throw new Error("Async error!");
            }, 1000);
        } catch (err) {
            console.log("Caught:", err.message); // ❌ won't run
        }

        👉 The error happens after the try/catch block finished, so it escapes.
        ✅ Fix → use try/catch inside the callback:

            setTimeout(() => {
                try {
                    throw new Error("Async error fixed!");
                } catch (err) {
                    console.log("Caught:", err.message);
                }
            }, 1000);

    3️⃣ Promises

        Errors in Promises are caught with .catch.

        Promise.reject("Promise failed")
            .catch(err => console.log("Caught:", err));


        👉 Output:
            Caught: Promise failed

    4️⃣ async/await with try/catch

        The best way to handle errors in async code.

        async function fetchData() {
            try {
                let res = await fetch("https://invalid.url"); // fails
                let data = await res.json();
                console.log(data);
            } catch (err) {
                console.log("Caught:", err.message);
            }
        }
        fetchData();


        👉 Output:
            Caught: fetch failed
