is javascript single threaded?
-------------------------------
JavaScript is a single threaded synchronous language.

🔹 Event Loop = Concept
	The event loop itself is just a mechanism inside JS runtime (browser or Node.js).
	It defines how synchronous code, microtasks, and macrotasks are scheduled and executed.

Promises, settimeout, setinterval, async-await are a way to handle asynchronous events.
